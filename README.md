# 🚀 Codveda Internship

This repository contains my work completed during the **Full Stack Development Internship at Codveda Technologies**.

The repository demonstrates various full-stack development concepts including REST APIs, authentication, database integration, frontend development, real-time communication using Socket.IO, and GraphQL.

---

## 📖 Project Overview

The internship consisted of multiple tasks designed to gradually build practical full-stack development skills. Each task focused on implementing features commonly used in real-world web applications.

---

## ✅ Tasks Completed

### 1️⃣ Environment Setup

* Configured Node.js and npm
* Initialized project structure
* Set up Git and GitHub version control
* Managed project dependencies

### 2️⃣ Simple REST API

* Built RESTful APIs using Node.js and Express.js
* Implemented CRUD operations
* Structured routes and API logic
* Tested endpoints using Thunder Client and Postman

### 3️⃣ Frontend with HTML, CSS and JavaScript

* Built a frontend using Vanilla HTML, CSS and JavaScript
* Implemented form handling
* Connected frontend with backend APIs
* Preserved this version as a reference implementation

### 4️⃣ Authentication & Authorization

* Implemented JWT-based authentication
* User registration and login functionality
* Protected routes using middleware
* Added role-based access support

### 5️⃣ Database Integration

* Integrated MongoDB using Mongoose
* Created database schemas and models
* Implemented database CRUD operations
* Enabled persistent data storage

### 6️⃣ Frontend with React

* Built a frontend using React and Vite
* Implemented component-based architecture
* Connected React frontend with backend APIs
* Used React Hooks for state management

### 7️⃣ Full Stack Application

* Connected frontend and backend applications
* Implemented complete user management flow
* Secured API communication using JWT
* Demonstrated end-to-end full-stack functionality

### 8️⃣ Real-Time Features with Socket.IO

* Implemented WebSocket communication using Socket.IO
* Displayed connected user count in real time
* Handled user connection and disconnection events
* Demonstrated real-time application concepts

### 9️⃣ GraphQL API

* Created a GraphQL server
* Defined GraphQL schema and queries
* Retrieved data using GraphQL Playground
* Demonstrated GraphQL alongside REST APIs
